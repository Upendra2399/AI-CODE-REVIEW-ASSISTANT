# 🤖 AI Code Review Assistant

An AI-powered web application that reviews source code and provides intelligent suggestions for improving code quality, readability, performance, and maintainability.

Built using **Python**, **Streamlit**, and **Groq LLM API**.

# 📌 Project Overview

The AI Code Review Assistant analyzes programming code using Generative AI and provides:

- 🐞 Bug Detection
- ✅ Best Practice Suggestions
- ♻ Refactoring Recommendations
- 📘 Documentation Improvements
- 💡 Improved Version of Code

This project helps developers write cleaner, more efficient, and maintainable code using AI-generated insights.

---

# ✨ Features

## 🐞 Bug Detection
Detects:
- Logical errors
- Syntax mistakes
- Runtime risks
- Common coding issues

---

## ✅ Best Practices
Suggests:
- Better coding standards
- Cleaner syntax
- Naming conventions
- Optimized approaches

---

## ♻ Refactoring Suggestions
Provides:
- Modularization ideas
- Code simplification
- Readability improvements
- Performance optimization

---

## 📘 Documentation Recommendations
Generates:
- Better comments
- Function descriptions
- Documentation improvements

---

## 📂 File Upload Support
Users can upload:
- `.py`
- `.java`
- `.cpp`
- `.js`
- `.c`

files directly into the application.

---

## 📥 Download Report
Users can download the AI-generated code review report as a text file.
